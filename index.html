Announcements

November 30, 2018

There are no more lecture or lab meetings. You have the choice to show up for the final exam on one of the following dates:

  • Monday, December 3 from 9:00 AM to 10:20 AM.
  • Friday, December 7 from 8:00 AM to 9:50 AM

The final exam is cumulative. Most questions will cover HTML and CSS. There will be a few questions on Javascript that relate to the Document Object Model. More than half the exam will ask questions similar to those from the midterm. The final exam will not be longer than the midterm. Good luck.

October 31, 2018

The midterm will be on Wednesday, November 7. It will cover HTML and CSS. For HTML, be sure to know how to create paragraphs, headings, lists, tables, links, and images. For CSS, familiarize yourself with the box model, display types, colors, and text formatting properties.

October 15, 2018

There will be a short quiz on Monday, October 22. You should be familiar with links, images, lists, and a little CSS to be prepared for it.

October 13, 2018

The Quizzes page has been added to this website. Blank copies and solutions to past quizzes can be found there.

October 1, 2018

There will be a 10 question quiz on Monday, October 8. It will cover material from the first week.

On Dr. Turner's Github site for this course, there is a link to a grade sheet for so that you may track which assignments you have gotten credit for. Email Dr. Turner a csusbdt@csusb.edu to request your grade sheet ID, include "CSE 322" in the subject.
